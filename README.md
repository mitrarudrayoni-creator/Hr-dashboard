🚀 Overview

The HR Analytics Dashboard is an interactive Power BI project designed to analyze workforce data and provide insights into employee performance, attrition, and organizational trends.

This dashboard helps HR teams and management make data-driven decisions to improve employee retention, engagement, and overall productivity.

🎯 Objectives
Monitor employee attrition and retention trends
Analyze workforce demographics
Track employee performance and engagement
Identify key factors affecting attrition
Support strategic HR decision-making
📊 Key Features
🔹 KPI Summary
Total Employees
Attrition Rate
Average Salary
Average Experience
Average Performance Rating
🔹 Attrition Analysis
Attrition by Department
Attrition by Gender (Stacked Column)
Attrition by Age Group
Attrition trends over time
🔹 Demographic Insights
Gender distribution
Age buckets
Education level
Location-wise analysis
🔹 Performance & Engagement
Engagement vs Performance (Scatter Plot)
Department-wise performance comparison
Experience vs Salary analysis
